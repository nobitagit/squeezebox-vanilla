Squeezebox.js is a minimal, ultra-light jQuery plugin for easely adding accordions to any website. It's built to retain semantics and to have the lowest impact possible on markup.

Setup

Squeezebox.js expects a very minimal structure to work and does not add any DIVs or HTML tags. It comes with predefined, overridable classes for its elements. Here's the minimum required layout the plugin needs:

<div class="accordion-1">
 <div class="squeezhead">
  header #1
 </div>
 <div class="squeezecnt">
  here goes the content
 </div>
</div>

In your js file hook to the wrapper element of the whole accordion and fire the plugin like so:

$('.accordion-1').squeezebox(); Here's the result:

A slightly more advanced usage

Even though the plugin comes with out-of-the-box, predefined classes for both the headers and the togglable content by no means you are bound to use those. Just pass them as options of the plugin.

In the following example we're also defining a different speed of the opening/closing animation. Another option passed makes sure that open folders are not automatically closed when a new one is opened.

Accordion with callbacks

Along with the set of option shown above the plugin offers two different callbacks you can hook to. One is fired when opening a folder, the other when closing it. Both expose the whole object created at init time. This allows for easy accessing to the elements such as the headers and the folders so we can toggle classes when the accordion is used.

Further details in the docs.

$('.accordion-3').squeezebox({
  // make sure you pass the element (el)
  // as a parameter so you
  // can gain acces to it
  onOpen: function(el){
   el.clickedEl.addClass('accordion_open');
 },
  onClose: function(el){
   el.clickedEl.removeClass('accordion_open');
 }
});

Docs

Here is a full overview of the settings the plugin allows.
headers
(property/string) the class of the headers of each folder
folders
(property/string) the class of the folders that get shown/hidden by the plugin
closeOthers
(property/boolean) if set to true only one folder per accordion can be open at one time. If set to false the user will be able to see the content of multiple folder at once.
closedOnStart
(property/boolean) if set to true the accordion will appear closed at page load.
animated
(property/boolean) if set to true the folders will be animating when opened. Set to false to avoid animation.
onOpen
(method/function) callback fired when opening a folder. By passing the whole element as a parameter (e.g. function(el) ) you will expose all the elements accessible with this API:
  • el.wrap: the wrapping div where the whole plugin fires
  • el.clickedEl: the heading that was clicked
  • el.relatedFolder: the related div that is toggled
onClose
(method/function) callback fired when closing a folder.

These are the default values that get used if no options are passed. It exposes the same API as the onOpen method.

$('.accordion-wrap').squeezebox({
 headers: '.squeezhead',
 folders: '.squeezecnt',
 closeOthers: true,
 closedOnStart: true,
 animated : true
});
